I'm a graduate student at Northwestern University studying programming languages. Right now, I'm working on making JavaScript and TypeScript more pleasant to write. Challenges working with those languages at scale abound; I'm interested in making the web a simpler place.

I also adore LISP and want some dialect of the language to spread. If you're interested in learning more, consider starting with Racket. I want as many people as possible to discover the joy of building programs by writing down the syntax tree that represents them directly.

Other interests include:

  • Package management systems
  • Symbolic AI
  • Web development
